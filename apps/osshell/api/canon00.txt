
 Canonical layouts

-------------------------------------------------
For os shell:

Task bar: (System bar)
[Start menu] [Desktop viwer] [Back] | X X X X ...

Start menu:    Display all the apps. Or maximize the selected.
Desktop viwer: Minimize all apps and display the desktop.
Back:          Revert interface to the last stage.

Recent button.
Home screen button.
Back button.

The task bar send messages to the display server 
when the user clicks on buttons and the display server 
sends the message to the active window.
The apps need to understand the messages generated by the taskbar.

The display server sends messages to the taskbar 
for creating and deleting icons, display the desktop, etc.
